# Portfolio

A macOS-inspired personal portfolio website built with Next.js and React.

## Features

- macOS boot sequence and welcome screen
- Interactive dock with hover animations
- Projects panel with detailed project information
- About Me section with skills and experience
- Responsive design with glassmorphism effects

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.
